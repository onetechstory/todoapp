## Task To Do Angular App

## Version Info
    Angular CLI: 8.3.26
    Angular: 8.2.14

## Tasks Performed

1. Setup Angular App with Routing, Styling as SCSS, Prefix as todo

    `ng new TaskToDo --routing --prefix=todo --style=scss`

2. Package Updation:

    `npm install`

3. Setup git

4. Development

5. Running the project

    `ng serve`

6. Building the project

    `ng build --prod`
